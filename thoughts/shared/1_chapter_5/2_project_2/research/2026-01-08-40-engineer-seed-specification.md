## Key Constraint Relaxation Scenarios Enabled

### Scenario 1: The Example Query
```typescript
{ requiredSkills: ['Kubernetes', 'Kafka'], minConfidenceScore: 0.9, minYearsExperience: 10, availability: 'immediate' }
```
- **0 results** with all constraints
- Relax experience to 8 → Priya (Kafka), Sofia (K8s) — still 0 (need both)
- Relax availability to two_weeks → Dmitri (has both!)
- Relax confidence to 0.75 → Anika (immediate, 10 years, has both)

{ timezone: 'Asia/*', minYearsExperience: 6, technicalDomain: 'Backend' }

## Evidence Requirements

| Evidence Type | Per Engineer | Total (40 engineers) |
